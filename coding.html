Coding

NOTE:
1. Either individual entries or a team of two is allowed.
2. The languages available are C, C++ (03, 11, 14), Java (7, 8, 9), Python (3.6.0, 2.7.9), and Go. The participants are free to choose any language and the choice can be changed anytime.
3. There will be 5 levels. Each level will consist of 1 question, the solution of which will be an integer. The participants will have to write a code that finds and prints the answer.
4. Once the participants successfully complete a level, they will be given access to the next level.
5. Scores shall be based on - the current level, time taken (starting from the start of the level till the correct answer is recorded), and number of wrong attempts.
6. The participants are allowed to run the code locally as many times as they want. Once the code is submitted, if the answer is valid, they can proceed to the next level, otherwise it will count as a wrong answer and points shall be deducted. The participants can submit as many times as they want.
7. Conclusions made by the governing body will be absolute & unchangeable.
